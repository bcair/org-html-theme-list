# 相关

为了方便自己的使用, 部分主题我 *clone* 了一个备份放在这个仓库, 这个仓库的备份可能不是最新的, 
如果你不介意的话, 也可以 *clone* 这个仓库.

其中来自 *Github* 的主题文件夹和原项目的名称相同.

来自 *Blog* 的样式按 `org-name` 的格式命名.
